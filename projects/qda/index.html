Statistical Data Analysis with Python

Questo progetto fornisce una panoramica delle tecniche di analisi statistica applicate alla qualità, con codice Python e teorie sviluppate nel contesto di modelli di controllo qualità.

Preview immagini

Argomenti trattati

  • Analisi descrittiva dei dati (mean, median, mode, variance, standard deviation)
  • Test di normalità (Shapiro-Wilk, Kolmogorov-Smirnov)
  • Test di ipotesi (t-test, z-test, chi-square test, ANOVA)
  • Analisi di correlazione e regressione lineare
  • Serie temporali: trend, stagionalità, smoothing
  • Control charts:
    • X-bar/R charts
    • X-bar/S charts
    • X-IMR charts
    • Probabilistic control charts
    • Trend and model based control charts
    • Fitted value and special causes control charts
    • Between group control charts
    • Multivariate control charts
    • EWMA control charts
  • Outlier detection e gestione dati mancanti
  • Analisi statistica multivariata (PCA, clustering)
  • Valutazione e interpretazione dei risultati per decisioni di qualità
